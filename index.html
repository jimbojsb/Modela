What is Modela?

Modela is an Object-to-Document Mapper for PHP and CouchDB. Modela allows you to build a domain model with PHP objects and persist them as documents in CouchDB.

What features does Modela Have?

  • Extremely light-weight HTTP access to CouchDB via raw PHP sockets
  • Extensible document base class for creating models
  • Autoloading of classes that won't conflict with other libraries
  • Robust API for defining and querying map-reduce views
  • Zend Framework application resource plugin
  • Zend Framework authentication adapter
  • Much more

Where and how do I get it?

Modela is hosted and maintained on GitHub. You can download the latest source version directly from GitHub if you like.

The preferred method for getting Modela is to install via PEAR. You can install Modela as follows: 

  # pear channel-discover mode.la
  # pear install modela/Modela-0.2.2
