Modela's PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover mode.la
Listing available packages: 
pear remote-list -c modela
Installing a package: 
pear install modela/package_name
Installing a specific version/stability: 
pear install modela/package_name-1.0.0
pear install modela/package_name-beta
Receiving updates via a feed: 
http://mode.la/feed.xml
